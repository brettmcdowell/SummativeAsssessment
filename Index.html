Malware Definition:

Malware, short for 'Malicious Software' is intrusive software that is intentionally designed to cause disruption, damage, or gain unauthorised access to a computer, server, client or computer network

The most common types of malware include computer viruses, computer worms, Ransomware, Keyloggers, Trojan horses, spyware and other examples of malicious software. Others include Fileless Malware, Spyware Adware, Rootkits, Bots, RAM scraper, and Mobile Malware

What are Viruses?

A computer virus is a type of computer program that, when executed, replicates itself by modifying other computer programs and inserting its own code. If this replication succeeds, the affected areas are then said to be "infected" with a computer virus

Every computer virus has a payload that performs an action. The threat actor can code any malicious activity into the virus payload, including simple, innocuous pranks that donâ€™t do any harm. While a few viruses have harmless payloads, most of them cause damage to the system and its data. There are nine main virus types, some of which could be packaged with other malware to increase the chance of infection and damage. The nine major categories for viruses on computers are: Boot Sector Virus, Web Scripting Virus, Browser Hijacker, Resident Virus, Direct Action Virus, Polymorphic Virus, File Infector Virus, Multipartite Virus, and Macro Virus

What are Worms?

A standalone malware computer program that replicates itself in order to spread to other computers. Often uses computer network to spread itself, relying on security failures on the target computer to access it

Computer worms often rely on vulnerabilities in networking protocols, such as File Transfer Protocol, to propogate. After a computer worm loads and begins running on a newly infected system, it will typically follow its prime directive: to remain active on an infected system for as long as possible and spread to as many other vulnerable systems as possible. The main types of computer worms are: Email Worms, File-Sharing Worms, Instant Messaging Worms, Crypto-Worms, and Internet Worms

Open Source Intelligence Definition:

Collection and analysis of data gathered from open sources (covert and publicly available) to produce actionable intelligence. Open Source Intelligence is used in national security, law enforcement, and business intelligence and is of value to analysts who use non-sensitive intelligence in answering classified, unclassified, or proprietary intelligence requirements across the previous intelligence disciplines

Open Source Intelligence traces its origins to the creation of the Foreign Broadcast Monitoring Service, an agency responsible for the monitoring of foreign broadcasts. The web browser is a powerful OSINT tool that provides access to numerous websites and both open source and proprietary software tools that are either purpose-built for open source information or which can be exploited for the purposes of either gathering of open source information or to facilitate analysis and validation to provide intelligence

What is Reverse Image Searching

Using Search Engines, you can quickly discover visually similar photos from around the web using Reverse Image Searching technology, utilizing content-based image retrieval (CBIR) query techniques. Uploading a photograph from your device or inputting the URL of an image, you can ask a search engine to locate and show you related images used on other websites, either those images that are exactly the same or the same but a different size, or those that contain similar looking items or people.

What is Metadata Search?

The best models of cameras allow you to make geotags (coordinates where the picture was taken), time (in EXIF format it is accurate only if the clock is set correctly on the device, but without reference to the world standard). Using different identifiers of metadata objects and through searching the metadata, you can possibly do the following: establish the person on the photo, determine the geographical location of objects, find out the time when the picture was taken, get information about the authorship of the picture (device and details about it), as well as data on subsequent transformations using picture editors. This is an extremely useful as it allows information to be ascertained about a subject with relative ease
